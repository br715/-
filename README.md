# -我的python文字遊戲 
   
(1)生死遊戲
由俄羅斯輪盤變形而成，規則:  
兩人各自7滴血，共用一把手槍，每發有為空包、實彈不明，你可以選擇射向自己或對方。  
此發若為空包彈:  
1.射向對方:可惜，將轉換手槍控制權。2.射向自己:很幸運，下一發仍是你的操作回合。  
此發若為實彈:  
1.造成1點傷害，不論射向對方或自己，都轉換手槍控制權。  
回合最後一發子彈決定的控制權對象，仍會延續到下一回合首發子彈。    
道具:  
每回合隨機發派兩個，可延續回合使用，改變戰局。  
雞腿:自身恢復1點生命值。  
藥水:自身恢復1點生命值或是扣除2點生命值。  
手銬:禁錮對方，使用後下一發子彈，無論射擊對象，無條件是自身回合，使用一次後消失。  
護盾:保護自己，受到傷害時將受到的傷害反彈，使用一次後消失。  
(這個世界只剩下伽利略的慣性運動，子彈撞到護盾後也不停下!如果雙方都使用護盾的話...)  
炸彈:下一發造成的+1傷害，可疊加。  
放大鏡:查看下一發空包與否。  
回合將重複直至某方血條歸0，遊戲結束，請用運氣和實力存活至最後一刻!
